radio

Endless generative music, synthesized one bar at a time with pure numpy and running entirely in your browser. Nothing loops: an unseeded RNG drives every choice, so what you hear has never played before and never will again.

starting up...

First load fetches the Python runtime and numpy (~10-15 MB), then caches them. Pause holds the current track; Stop resets it, so the next start is a fresh, never-before-heard take. Switching channels crossfades; Stop then a channel is a hard cut.
